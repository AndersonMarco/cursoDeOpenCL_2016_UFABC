Tutorial de OpenCL

De 12/12 a 16/12 será ministrado um tutorial, presencial e a distância, sobre a API de programação paralela OpenCL. Por Anderson Gonçalves Marco (mestrando em Ciência da Computação na UFABC) e por Mirelle Alves de Freitas (aluna do Bacharel em Ciência e Tecnologia).

Onde

  • Presencial
    UFABC Câmpus Santo André (Av. dos Estados, 5001, Santo André) - Bloco A, Torre 2, Laboratório L408-2.

  • Distância
    Acesse o canal do youtube e a salas virtuais um e dois do Rocket.Chat.

Informações

  • Valor: gratuito!

  • Horário
    Das 14:00 as 16:30 aulas junto com alguns exercícios simples.
    Das 16:30 às 18:00: resolução de dúvidas complexas, envolvendo “paralelização” de códigos desenvolvidos pelos alunos e a instalação do OpenCL no Windows.

  • Requisitos
    Conhecimento básico sobre a linguagem C (coisas como printf, malloc, ponteiros, structs e criação de funções). Python é um extra.

  • Informações adicionais
    Para os alunos à distância, é necessário ter o Linux instalado para resolver os exercícios. Uma VM é suficiente. Eu utilizarei a distribuição Linux Mint LMDE 2. Também pode instalar o Fedora 24 ou o Debian 8.

Conteúdo programatico

Os seguintes tópicos vão ser abordados:

  • Historia das GPUs

  • Introdução ao paralelismo

  • OpenCL com C

  • Integração de OpenCL com OpenGL

  • Profiling de aplicações OpenCL

  • OpenCL com python

  • Aritmética de ponto fixo de 128 bits
